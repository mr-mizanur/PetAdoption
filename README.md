# Pet Adoption Registry Database

## Purpose
A full-stack MERN Pet Adoption Platform designed to connect pet owners/shelters with animal lovers. This platform streamlines the adoption process, allowing users to browse pets, submit adoption requests, and manage their own profiles while ensuring a secure adoption experience.

## Live URL
[[https://pet-adoption-theta-ten.vercel.app/](https://pet-adoption-theta-ten.vercel.app/)]

## Key Features
- **Global Registry:** Explore a diverse range of pets (dogs, cats, birds, rabbits, etc.) with real-time search, filter, and sorting capabilities.
- **Secure Authentication:** User registration, Login, and Google Login with JWT-based protection stored in HTTPOnly cookies.
- **Dashboard for Pet Owners:** A dedicated private dashboard to add, update, and delete pet listings, and approve/reject adoption requests.
- **Adoption Request Management:** Users can submit and track the status of their adoption requests in real-time.
- **Responsive & Dynamic UI:** Built with Next.js, Tailwind CSS, and Framer Motion for a smooth, user-friendly experience across all devices.

## Technology Stack
- **Frontend:** Next.js (App Router), Tailwind CSS, Framer Motion, React Icons, Axios.
- **Backend:** Node.js, Express.js, MongoDB (Mongoose), JWT, Cookie-Parser.
- **Deployment:** Vercel (Client) and Render/Vercel (Server).

## NPM Packages Used
- **Client:** `next`, `framer-motion`, `axios`, `react-hot-toast`, `react-icons`.
- **Server:** `express`, `mongoose`, `jsonwebtoken`, `cookie-parser`, `cors`, `dotenv`.
